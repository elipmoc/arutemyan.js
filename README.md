# Getting Start

```
npm i arutemyan
```

# Usage

```
const Arutemyan = require('arutemyan');
const a = new Arutemyan();

console.log(a.age);
console.log(a.hiraganaName);
console.log(a.name);
console.log(a.isThursday);
console.log(a.generateMokuMoku('木曜日になった'));
```
